# Unused assets report

This list includes asset files under `assets/` that are not referenced by string literals in Dart files (excluding `lib/generated/assets.dart`). It also treats dynamic flag/translations/data loads and `sounds/` paths as used, and scans the repo for other text references (e.g. pubspec configs).

## Commands used
- `find assets -type f`
- `rg -n "assets/" lib main.dart test`
- `rg -n "assets/flags" lib main.dart test`
- `rg -n "correct\.mp3|sounds/|\.mp3'|\.wav'" lib main.dart test`
- `rg --fixed-strings <asset-path> . (repo-wide scan excluding assets/unused, docs/unused-assets.md, lib/generated/assets.dart)`
- `python (asset usage analysis script)`
